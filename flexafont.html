Flexafont Web App

Problem

Text on websites is often difficult to read for people with vision problems

• Come up with an app that can help alleviate this issue

In a nutshell

app that solves text-reading issues for users

My take

app that allows the user to customize the font of text on a website for better legibility; app also allows for designers and developers to test out new fonts on already existing websites

Why

everyone should have access to publicly available information they need, including those with disabilities

Who

solo project

Technologies

Sketch, Balsamiq

Wireframes These interactive wireframes portray the app's navigation and options. Users can log in, create, save, load, and customize fonts.


App name and tagline
Flexafont
Stay focused on the reading at hand with flexible font choices.

Problem and details
Have you ever found it difficult to stay focused on a website’s text content because the font is unreadable? Font that is too small, the wrong style, or has poor color contrast makes it taxing on the reader. Whether you have attention problems and need your current selection to stand out, color blindness, or other vision complications, Flexifont allows the text you’re reading to display on your terms.

- Open app > Select your text parameters (name, size, style, color, background) > Minimize app (do not quit) > Highlight desired text in browser to be changed > Button will automatically populate at end of selection > Click button to apply parameters chosen in app to website text (text will only be changed on client side, not on actual website)

Main functionalities
1) Select typeface name 2) Select text size 3) Select text color 4) Select text background color 5) Select text style (regular, bold, italic, etc.)
(provides a way to change a website’s text to display different aesthetics)

How does it work?
- Open app > Select your text parameters (name, size, style, color, background) > Minimize app (do not quit) > Highlight desired text in browser to be changed > Button will automatically populate at end of selection > Click button to apply parameters chosen in app to website text (text will only be changed on client side, not on actual website)

Who benefits from this app?
- people with attention problems, color blindness, or other vision complications

Why would someone use this?
1) For people with ADHD, it is difficult getting through long articles on the web; there is a need to be able to select different text background colors to stay focused on 1 paragraph at a time

2) For people with astigmatism, text appears distorted; there is a need for bigger text and the ability to choose specific fonts

3) For people with colorblindness, text may not always appear distinct enough from background; there is a need to be able to choose text and text background colors

4) For people with Presbyopia (the loss of near-sighted focusing ability that occurs with age), text appears fuzzy; there is a need to be able to choose text size and change text style to bold
